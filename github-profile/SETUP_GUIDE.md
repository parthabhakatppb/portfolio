# 🚀 How to Set Up Your Animated GitHub Profile (`parthabhakatppb`)

To make your GitHub Profile (`https://github.com/parthabhakatppb`) animate and look just like your futuristic AI/ML portfolio, follow these 3 quick steps:

---

### Step 1: Create Your Profile Repository on GitHub
1. Go to `https://github.com/new`
2. In the **Repository name** field, type exact name: **`parthabhakatppb`**
   *(You will see a green banner: "You found a secret! parthabhakatppb/parthabhakatppb is a special repository that you can use to add a README.md to your GitHub profile.")*
3. Make sure **Public** is selected.
4. Check **Add a README file** and click **Create repository**.

---

### Step 2: Upload Your Animated Banner & README
Inside your new `parthabhakatppb/parthabhakatppb` repository:
1. Upload the custom animated SVG banner we generated:
   - Copy `public/github-banner.svg` from your portfolio directory.
   - Upload it directly into your root folder on GitHub as `github-banner.svg`.
2. Replace the `README.md` contents:
   - Open `README.md` in GitHub and click the **Edit (pencil icon)** button.
   - Copy everything inside `github-profile/README.md` from this folder and paste it into GitHub's `README.md`.
   - Click **Commit changes**.

---

### Step 3: Enable the Neon Contribution Snake Animation (`snk`)
1. In your `parthabhakatppb/parthabhakatppb` repo on GitHub, click **Add file -> Create new file**.
2. In the file path box, type: `.github/workflows/snake.yml`
3. Copy everything from `github-profile/snake.yml` and paste it inside.
4. Click **Commit changes**.
5. Go to the **Actions** tab on GitHub, click on **Generate Contribution Snake Animation** on the left, and click **Run workflow** -> **Run workflow**.

In ~60 seconds, GitHub will generate your custom dark-mode neon electric cyan/violet snake eating through your commit grid, and your entire profile will glow!
